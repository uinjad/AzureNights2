# Changelog

All notable changes are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/), and the project aims to follow
semantic versioning once it reaches v1.

## [Unreleased]
### Added
- Five connected zones gated by zone bosses; defeating a boss unseals the exit.
- Victory condition and screen on clearing the Obsidian Keep.
- Passive HP/MP regeneration while exploring (replaces campfires).
- Loot drops: rare gear (5%) and common potions (20%, HP and MP).
- Custom hero name entry and save loading on the title screen.
- Cross-platform release binaries via GoReleaser.

### Changed
- Enemies are tuned per zone instead of scaled by level, removing one-shot swings.
- Quests and class advancement now live in the character menu (`c`).
